# Sim Name Generator

> A simple name generator for The Sims, built with Node.js and EJS.

## üöÄ How to Run This Project

To get a local copy up and running, follow these simple steps.

### Prerequisites

You must have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/your-username/sim-name-generator.git](https://github.com/your-username/sim-name-generator.git)
    ```
2.  Navigate to the project directory:
    ```bash
    cd sim-name-generator
    ```
3.  Install the NPM packages:
    ```bash
    npm install
    ```
4.  Run the application:
    ```bash
    npm start
    ```
    *(Or `nodemon index.js` if you are using nodemon)*

5.  Open your browser and go to `http://localhost:3000`.
